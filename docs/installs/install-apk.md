# Install APK

Install APK opens a local APK file you choose and hands it to Android's normal package installer.

## Where to find it

Main menu or install tools -> **Install APK**.

## What it does

Adult Game Manager lets you pick an APK from local storage, then starts Android's install flow. Android decides whether the APK can be installed and may ask you to allow installs from that source.

## What it does not do

The app does not download APKs for you, bypass Android security prompts, or silently install packages.
