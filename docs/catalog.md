# Catalog

The Catalog tab lets you search source catalogs even for games that are not currently installed.

![Catalog](screenshots/catalog-main.png)

## Filters

Use filters for installed state, source, platform, status, engine, rating, and sort mode. Tag search supports typed tag filters.

![Catalog tag filter](screenshots/catalog-tag-filter.png)

## Opening a page

Open a catalog entry to view details and launch the related page in your browser. The app does not download game files automatically.

## Source catalogs

Adult Game Manager uses a source-neutral catalog model. A source can be F95Zone, AdultGameWorld, or another configured catalog feed. Catalog entries keep their source name and canonical page URL so source-specific links still open the right site.
