# Catalog

The Catalog tab lets you search the app's catalog even for games that are not currently installed.

![Catalog](screenshots/catalog-main.png)

## Filters

Use filters for installed state, status, engine, rating, and sort mode. Tag search supports typed tag filters.

![Catalog tag filter](screenshots/catalog-tag-filter.png)

## Opening a page

Open a catalog entry to view details and launch the related page in your browser. The app does not download game files automatically.
