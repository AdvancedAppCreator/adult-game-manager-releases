# Refresh installed games

Refresh from catalog updates installed-game rows using the current catalog data.

## Where to find it

Main menu -> **Refresh from catalog**.

## What it does

Adult Game Manager compares local games against cached catalog entries, updates match candidates, refreshes version/update state, and keeps source-specific links attached to matched games.

## Before refreshing

Sync catalogs first if you have not refreshed them recently. Refreshing with stale catalog data can leave update state unchanged.

## If a row stays unmatched

Use [Manual search](../mapping/manual-search.md), [Paste a page URL](../mapping/paste-url.md), or [Review unmapped games](review-unmapped.md).
