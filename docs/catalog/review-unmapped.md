# Review unmapped games

Review unmapped games walks through local rows that are not yet linked to a catalog entry.

## Where to find it

Catalog tools -> **Review unmapped games**.

## What it does

The review flow shows likely matches and lets you accept a match, search manually, paste a page URL, skip, or mark the row as not in the catalog.

## Tips

- Sync catalogs before starting a review pass.
- Use source names and URLs to avoid accepting a same-title match from the wrong source.
- Skip uncertain rows rather than forcing a weak match.
