# Auto-backups

Auto-backups protect app state before risky operations when the feature is available for that workflow.

## What they are

An auto-backup is a local backup created by Adult Game Manager before an operation that may change mappings, imports, or JoiPlay-related state.

## Where they are saved

Auto-backups are saved to local app/document storage according to Android file-access rules and app configuration.

## Recommendation

Use manual export before major changes even if auto-backups are enabled.
