# Export and import backup

Backups preserve Adult Game Manager state such as mappings, hidden/not-in-catalog flags, and personal tracking data included by the app.

## Where to find it

Menu -> **Backup & config ... -> Export backup** or **Import backup**.

## Export before

- Moving to a new device.
- Testing prerelease builds.
- Large matching or cleanup sessions.
- Changing advanced config.

## Import

Choose a backup file and review the restored state. Imports affect app data, not installed APKs or game files.
