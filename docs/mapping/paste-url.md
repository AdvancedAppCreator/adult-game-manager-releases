# Paste a page URL

Paste URL matching links a local game to a known source page when search is not enough.

## Where to find it

Open the manual match/edit dialog for a game and use the URL field when available.

## What it accepts

Use a normal source page URL, such as an F95Zone thread URL or another configured source URL. Adult Game Manager stores the source-aware link so the row can open the right page later.

## Tips

- Paste the canonical page, not a temporary search-result URL.
- If the app cannot parse the URL, use manual search or refresh the source catalog first.
