# Manual search

Manual search lets you choose the correct catalog entry when auto-match is missing or wrong.

## Where to find it

Open an installed game row and choose the match/edit action, or use the review-unmapped flow from the Catalog tools.

## What it does

Search by title, compare source names and page URLs, then apply the best match. The app stores the selected source entry for that local game.

## Good search terms

- Use the shortest distinctive part of the game title.
- Remove version numbers, platform tags, and punctuation.
- Try alternate spellings if the catalog source uses a different title.

## If no result is correct

Use [Paste a page URL](paste-url.md) if you know the exact source page, or mark it as [not in catalog](not-in-catalog.md) if the local app should be ignored.
