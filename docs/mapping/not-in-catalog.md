# Mark not in catalog

Marking a row as not in catalog tells Adult Game Manager not to keep suggesting matches for that local app or game.

## Where to find it

Open the match/edit action for a row, or use the review-unmapped workflow.

## When to use it

- The installed app is not an adult game.
- The game is private, modified, or not listed in any configured source.
- Auto-match repeatedly suggests unrelated entries.

## Undo

Open the row's match/edit action again and clear the not-in-catalog state if you want it to be considered for matching later.
