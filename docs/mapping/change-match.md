# Change a wrong match

If a local game is linked to the wrong catalog entry, edit the match instead of deleting the game.

## Where to find it

Open the game row and choose the match/edit action.

## What to do

1. Clear the incorrect match if needed.
2. Search for the correct title or paste the correct page URL.
3. Confirm the replacement match.
4. Refresh from catalog to update the displayed version and source link.

## Why it matters

Wrong matches can show the wrong update state and open the wrong page. Source-aware matching keeps the selected source and page URL attached to the local game.
