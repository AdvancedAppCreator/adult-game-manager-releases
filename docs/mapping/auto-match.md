# Auto-match

Auto-match links installed games to catalog entries using known package names, titles, and normalized name matching.

## Where to find it

Use **Refresh from catalog** from the main menu, or sync the catalog and then refresh installed games.

## What it does

Adult Game Manager compares local APK and JoiPlay names with the current source catalogs. When a confident match is found, the row can show the source page, known catalog version, and update state.

## Source-aware matching

Matches preserve their source. A game matched to AdultGameWorld opens the AdultGameWorld page; a game matched to F95Zone opens the F95Zone page.

## Tips

- Sync catalogs before judging match quality.
- Use manual search for alternate titles, abbreviations, or games with very generic names.
- Mark non-game apps or unrelated entries as not in the catalog so they stop distracting you.
