# Install game in JoiPlay

Install game in JoiPlay helps send a local archive or folder through the JoiPlay install/import flow.

## Where to find it

Menu -> **JoiPlay ... -> Install game in JoiPlay**.

## What it does

Choose a local game archive or file. Adult Game Manager prepares the handoff and uses configured JoiPlay settings to help install or update the game.

## Safety

Read prompts before replacing an existing game folder. When an operation may overwrite data, keep a backup of the original game folder or JoiPlay backup.
