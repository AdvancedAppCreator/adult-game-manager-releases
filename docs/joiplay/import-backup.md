# Import .joiback backup

Importing a `.joiback` file lets Adult Game Manager list and track JoiPlay games.

## Where to find it

Menu -> **JoiPlay ... -> Import JoiPlay backup**.

## Workflow

1. Export a `.joiback` backup from JoiPlay.
2. In Adult Game Manager, choose **Import JoiPlay backup**.
3. Select the backup file.
4. Review imported games on the main screen.

## Notes

The import reads local metadata from the backup. It does not download game files.
