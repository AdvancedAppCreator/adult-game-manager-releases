# Probably unused folders

Probably unused folders helps find JoiPlay folders that do not appear in a fresh JoiPlay backup.

## Where to find it

Menu -> **JoiPlay ... -> Probably unused JoiPlay folders**.

## What it does

Adult Game Manager compares known imported JoiPlay games with folders in configured locations and highlights folders that may no longer be referenced.

## Be careful

This is a cleanup aid, not proof that a folder is safe to delete. Check folder contents before removing anything.
