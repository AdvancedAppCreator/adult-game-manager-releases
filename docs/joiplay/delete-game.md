# Delete a JoiPlay game

Delete actions remove a local JoiPlay game folder only after you confirm the operation.

## Where to find it

Open a JoiPlay game row and use the delete/remove action when available.

## Safety

Deletion affects local files. Export a JoiPlay backup or copy the folder elsewhere before deleting if you may want to restore it.

## If deletion fails

Android storage restrictions or file locks can prevent deletion. Close JoiPlay, retry, or remove the folder with your file manager.
