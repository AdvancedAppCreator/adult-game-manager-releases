# Main screen

The main screen shows installed APKs and imported JoiPlay games with their current match/update state.

![Main screen](screenshots/main-screen.png)

## Top actions

Open the main menu from the top-right action button.

![Main menu](screenshots/main-menu.png)

Common actions include:

- **Refresh from catalog** - checks catalog matches and update status.
- **Check for app update** - checks whether a newer Adult Game Manager APK is available.
- **JoiPlay ...** - opens JoiPlay install/import/settings tools.
- **Backup & config ...** - opens backup, restore, and config tools.
- **Help ...** - opens documentation, About, support, and diagnostics.

## Filtering

Use the search field and filter chips to narrow the list. Catalog tag filters use `tag:` terms, such as `tag:renpy` or multiple tags in the same query.
