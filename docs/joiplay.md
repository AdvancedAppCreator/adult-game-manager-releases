# JoiPlay

JoiPlay tools help track and install local Ren'Py, RPG Maker, HTML, Tyrano, and similar games.

![JoiPlay settings](screenshots/joiplay-settings.png)

## Where to find it

Menu -> **JoiPlay ...**

## Common actions

- **Install game in JoiPlay** - select a local game file/archive and send it through the JoiPlay install flow.
- **Import JoiPlay backup** - import a `.joiback` file so Adult Game Manager can list your JoiPlay games.
- **JoiPlay settings** - configure paths and behavior used by JoiPlay helpers.
- **Probably unused JoiPlay folders** - review folders that do not appear in a fresh backup.

Archive install/upgrade actions create safety prompts and backups where possible. Review prompts before replacing a game folder.
